# Tribute Hub 🕯️

An elegant tribute website celebrating visionaries, artists, innovators, explorers, and pioneers who shaped our world.

---

## ✨ Features

- Responsive design for desktop and mobile devices
- Interactive category cards with JavaScript navigation
- Individual tribute counters using `localStorage`
- Confetti effect when paying tribute
- Dark and elegant UI inspired by museum and memorial websites
- Unique visual themes for different fields

---

## 🌍 Categories

- 🔬 Science — Albert Einstein
- 🤖 Artificial Intelligence — Alan Turing
- 📚 Literature — William Shakespeare
- 🎼 Music — Ludwig van Beethoven
- 🏅 Sports — Muhammad Ali
- 🚀 Space Exploration — Neil Armstrong
- 💡 Technology — A. P. J. Abdul Kalam
- 🎬 Cinema — Charlie Chaplin

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- Local Storage API
